Calls to action should be used for short links to tasks that a user can do next. For example, a link to a licence application service. They create a tinted box which highlights the task.

To create the call to action box, use $CTA above and below the link. For example:

```
$CTA
You can [read the content and publishing guidance](https://guidance.publishing.service.gov.uk/).
$CTA
```
