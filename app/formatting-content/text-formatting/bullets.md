To create a bullet point list:

* use asterisks `*` or hyphens `-` to make bullets 
* make sure there is a space after the asterisk or hyphen
* leave an empty line space before the bullets start, and one after

For example:

```
Leave a line below this:

* then an asterisk for a bullet
- or use a hyphen
```

See the [GOV.UK style guide for the style and punctuation rules on using bullet points](/writing-to-gov-uk-standards/style-guides/a-to-z-style-guide/#bullet-points-and-steps).
